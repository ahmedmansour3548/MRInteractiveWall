    public Admin(): Object source 

    import {Admin} from 'mrinteractivemural.client/src/Admin.js'

    Hook that manages the Admin page and manipulating AR content. This page provides a method to add, edit, and delete models, as well as define associations between lab members and models.

    public AR(): Object source 

    import {AR} from 'mrinteractivemural.client/src/AR.js'

    Hook that manages the AR scene's state and interactions. This page is where the main AR experience occurs. It utilizes Three.js and A-frame to create an WebAR window.

    public Home(): Object source 

    import {Home} from 'mrinteractivemural.client/src/Home.js'

    A hook that manages the state and behavior for the Home component. It handles navigation, interaction, and the background effects.

    State Variables and Functions

    NameTypeDescription
    isDrawerOpen Boolean

    State to manage the open/close status of the drawer component.
    toggleDrawer Function

    Function to toggle the drawer's open/close status.
    viewportSize Object

    State to manage the viewport size for background effects. Contains width and height properties.
    setViewportSize Function

    Function to set the viewport size state.
    squares Array

    State to store the calculated squares for the component, initially populated by calculateSquares function.
    setSquares Function

    Function to set the squares state.

    public calculateSquares(): Array<Object> source

    A callback that calculates the number of squares based on the current viewport size. It divides the viewport into a grid of squares, each with a side length defined by squareSize.

    Return:

    {squares: Array<Object>}: An array of squares Objects, each with a unique color.

    Return:

    Object

    The state values and functions neccessary to interface with HomeComponent.
