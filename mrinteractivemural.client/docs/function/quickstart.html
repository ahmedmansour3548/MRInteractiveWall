    Requirements

    To run the project, ensure the following is installed on your computer:

      Frontend

    • React 17.x or higher
    • Node.js w/ npm
    • A WebAR-compatiable browser

      • Backend

    • .NET 5 or .NET Core 3.1 LTS & SDK
    • MySQL 8.0 or higher

      • Hardware

    • A webcam or mobile phone with a camera

      • Optional:

    • MySQL Workbench
    • Visual Studio 2022 (To use preset configuration parameters)

    Basic Usage

    Visual Studio

    The project contains the necessary configuration to start the servers using Visual Studio. Open the project and click the green 'Run' button to start the process.

    Manual

    If spinning up the servers manually, you will have to perform the following actions within the root project directory:

    1. dotnet run - Starts the .NET server on port 7121
    2. npm start - Starts the Vite server on port 5173
    3. Start the local MySQL server on port 3306. Note that the configuration (schema, username and password) for the server must match exactly what is found in appSettings.json

    Once the servers are running, the webpage will open automatically.
